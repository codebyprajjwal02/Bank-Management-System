# Bank-Management-System
This project is a Java-based Bank Management System with an ATM Simulator that replicates the core functionalities of a real-world banking ATM. The system allows users to perform essential banking operations such as account creation, deposits, withdrawals, PIN management, and transaction tracking through a simulated ATM interface.
🚀 Features
Account Registration / Opening New Bank Account
Secure Login using Card Number and PIN
Cash Deposit and Withdrawal functionality
Balance Inquiry
Mini Statement for recent transactions
PIN Change feature
Transaction history tracking
Simple ATM-style user interface

🛠️ Technologies Used
Programming Language: Java
GUI: Java Swing / AWT
Database: MySQL
Database Connectivity: JDBC
IDE: Vs Code

📌 Project Objective
The goal of this project is to simulate how an ATM banking system works internally, providing hands-on experience with Java programming, database integration, and transaction management while building a real-world banking application.

🎯 Learning Outcomes
Implementation of Object-Oriented Programming (OOP) concepts
Working with Java GUI frameworks
Handling database operations using JDBC
Understanding ATM transaction workflows
